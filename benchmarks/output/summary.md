## Running Benchmarks

### Basic Usage

```bash
# Run all benchmarks
sentinel run

# Run benchmarks in parallel
sentinel run --parallel

# Output as JSON
sentinel run --json

# Dry run (don't write files)
sentinel run --dry-run

# Custom output directory
sentinel run --output /path/to/output
```

### Output Files

Results are written to:
- `benchmarks/output/raw/` - Individual JSON result files
- `benchmarks/output/summary.md` - This summary file
- `benchmarks/output/combined_*.json` - Combined results file
